# 续写生成（NDJSON，ResultStructure）

## 输出目标
- 目的：在当前会话语境下，续写生成 1–2 条中文回复建议
- 仅输出 NDJSON（每行一个 JSON 对象），不得输出任何解释或其他文本
- 严格符合 ResultStructure 字段结构（RFC 8259 合规）

## 输入参数
- 情感指数：{emotionalIndex}
- 情绪标签：{emotionalLabels}

## 生成约束
- 第一人称视角（“右侧（我）”的语气）
- message 为中文短句，尽量≤15字，口语化、自然、真人化表达
- 根据情感指数控制强度与表情符号数量（最多 1–2 个）
- 根据情绪标签调整风格与语气（如：温柔/直接/幽默/克制等）
- 避免复读历史表达与语气词，保持新鲜度与多样性
